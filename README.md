# 📘 Academic Writing Journey

**B2 → C1 · 6 Hours · 2,500 XP**

A fully gamified, asynchronous self-paced unit designed to move learners from B2 to C1 in academic English. Built using the **SAM + Casañ-Pitarch IPO instructional model**, combined with ESA and CLIL (4Cs). Created with Claude AI + Canva LMS.

🌐 **Live site:** https://profeanaeslavaedtech.github.io/Gamification/

---

## 🎯 Learning Objectives

- Master the **Swales 5-Move Abstract Structure** (M1–M5)
- Apply **Hyland's Metadiscourse framework** (Interactive + Interactional markers)
- Build academic vocabulary using the **Academic Word List (AWL)**
- Develop reading, writing, listening, and speaking skills in academic register
- Progress from Curious Student → Q1 Master Scholar

## 🛠️ Technical Stack

- **Single-file app:** `index.html` (~1,450 lines)
- **CSS framework:** Tailwind CSS (CDN)
- **Icons:** Font Awesome 6.5
- **Font:** Bebas Neue (Google Fonts)
- **Escape room:** Embedded via iframe from `https://profeanaeslavaedtech.github.io/escaperoom/`
- **Hosting:** GitHub Pages — branch `main`, root folder `/`

---

## 🚀 How to Use (Student Instructions)

1. Open the live site: https://profeanaeslavaedtech.github.io/Gamification/
2. Click **🎬 Watch Intro Video** to see the course introduction.
3. Click **Show Full Academic Overview** to read the full course description.
4. Complete levels sequentially — each level unlocks the next.
5. After Level 3, enter **The Abstract Heist** bonus escape room to unlock Level 6.
6. Submit your work using the tools indicated in each level.
7. Complete Level 6 to claim your **Diamond Crown of Publication** and watch the farewell celebration.

---

## ✏️ How to Edit (Teacher Instructions)

1. Go to https://github.com/ProfeAnaEslavaEdTech/Gamification
2. Click `index.html` → pencil icon (Edit)
3. Make your changes
4. Select **Commit directly to `main`**
5. GitHub Pages auto-deploys in ~1 minute
